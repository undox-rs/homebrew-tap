# Undox-rs Tap

## How do I install these formulae?

`brew install undox-rs/tap/<formula>`

Or `brew tap undox-rs/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "undox-rs/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
